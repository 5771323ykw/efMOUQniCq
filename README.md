# 人事管理系统简介 java135

## 项目概述

本人事管理系统是基于Spring Boot和Vue技术栈开发的，适用于企业内部员工信息与考勤管理。系统实现了管理员与员工两种角色的功能需求，提供了一站式的个人信息、考勤记录和薪酬管理。

## 技术栈

- 后端：Spring Boot
- 前端：Vue 2 + ElementUI
- 运行环境：Node.js 14
- 数据库：MySQL 5.7及以上
- JDK版本：1.8

## 功能模块

### 管理员角色

- **个人中心**：管理员可以查看和管理个人相关信息。
- **部门信息管理**：支持部门信息的添加、修改、删除操作。
- **员工信息管理**：支持员工信息的添加、修改、删除操作。
- **考勤信息管理**：管理所有员工的考勤情况。
- **上班记录管理**：管理员工的上班打卡记录。
- **下班记录管理**：管理员工的下班打卡记录。
- **加班记录管理**：管理员工的加班记录。
- **员工薪酬管理**：负责对员工薪酬进行管理。

### 员工角色

- **个人中心**：员工可以查看和管理个人相关信息。
- **考勤信息管理**：员工可以查看和管理个人的考勤信息。
- **上班记录管理**：员工可以查看个人的上班记录。
- **下班记录管理**：员工可以查看个人的下班记录。
- **加班记录管理**：员工可以查看个人的加班记录。
- **员工薪酬管理**：员工可以查看和管理个人的薪酬信息。

## 核心亮点

- **统一管理**：系统整合了员工从个人信息到工作记录的全流程管理。
- **权限分离**：根据不同角色划分清晰的管理权限，确保数据安全。
- **技术成熟**：基于成熟的Spring Boot和Vue技术栈，保证系统稳定性和可扩展性。
